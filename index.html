Configuration

Deploy worker.js to Cloudflare Workers, then enter your Worker URL below. The worker handles both AI nudges (Anthropic) and email sending (Resend).

1. npm install -g wrangler && npx wrangler login
2. npx wrangler deploy worker.js --name trilogy-care-nudge
3. npx wrangler secret put ANTHROPIC_API_KEY
4. npx wrangler secret put RESEND_API_KEY
5. npx wrangler secret put FROM_EMAIL
